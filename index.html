About the Project

Bridging Technology
with Accessibility

Designed with empathy, built with precision — a smart room system that adapts to the user, not the other way around.

🚩 Problem Statement

Mobility-challenged individuals — including students with physical disabilities and elderly residents — often struggle with conventional switches and controls. Standard room appliances are inaccessible, fostering dependence and reducing quality of life. There is a critical need for an intuitive, remote-operated control system that restores independence.

💡 Our Solution

A fully IoT-integrated Smart Room system that allows users to control lights and fans via a web dashboard, voice commands, and mobile devices. Built on ESP8266/ESP32 with Firebase cloud sync, it provides real-time feedback and emergency alert capabilities, all within a beautifully accessible interface.

Core Features

Everything You Need
to Control Your Space

Six powerful features designed around accessibility, comfort, and safety.

💡

Remote Light Control

Toggle lights ON or OFF from anywhere using the web dashboard, voice commands, or mobile interface — no physical switches required. Designed with large, accessible buttons for ease of use.

🌀

Fan Speed Adjustment

Control fan speed across 5 levels with smooth controls. The system uses PWM-regulated relays to precisely adjust fan RPM, providing tailored comfort for every user preference.

📡

Real-Time Status Monitoring

Live device status is synchronized via Firebase, displaying color-coded indicators (Green = ON, Red = OFF) on the dashboard. Status updates reflect within milliseconds of any change.

🎙️

Voice Assistant Integration

Control all appliances hands-free using the browser-based Web Speech API. Simply say "Turn on the light" or "Set fan to level 3" — fully accessible voice control built right in.

🚨

Emergency Alert Feature

A dedicated SOS button immediately sends an alert to caregivers via Firebase. Designed for rapid response in critical situations with a single tap or voice command.

📱

Mobile-First Dashboard

A fully responsive web dashboard accessible from smartphones, tablets, and desktops. High-contrast mode, large touch targets, and accessibility-first design ensure usability for everyone.
